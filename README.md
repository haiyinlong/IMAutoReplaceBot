# IMAutoReplaceBot
IM 自动回复机器人，通过RPA来实现

## 安装步骤
1. 创建Python 虚拟环境

```bash
python3 -m venv venv
```
2. 激活虚拟环境
windows
```bash
venv\Scripts\activate
```

linux
```bash
source venv/bin/activate
```
3. 安装依赖包
```bash
pip install -r requirements.txt
```
4. 修改配置文件config.json
5. 运行程序
```bash
python main.py
```
